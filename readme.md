## 💻 How to run
#### 1.Clone
  git clone https://github.com/lenamiroux/Proffy

#### 2. Find directory
> cd Proffy

#### 3. Download dependecies
>  npm install

#### 4. Run server
>  npm run dev

#### 5. Open on browser
>  Open http://localhost:5500/ on your browser
